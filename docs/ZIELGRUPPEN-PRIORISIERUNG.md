## 🎯 Detailprofil je Branche

### Rang 1: GaLaBau / Landschaftsbau — Score 4.20 🔴

**Empfehlung: Nächster Zielmarkt — sofort starten**

| Kennzahl | Wert |
|---------|------|
| KMU-Betriebe DACH | ~45.000 (DE: 35.000 · AT: 6.000 · CH: 4.000) |
| TAM | 150–250 Mio. € |
| Ø Betriebsgröße | 8–25 Mitarbeiter |
| Digitalisierungsgrad | 🔴 Sehr niedrig (<20%) |
| Saisonalität | März–November (Hauptsaison) |
| Durchschnittlicher SaaS-Preis | 200–800 €/Monat |

**Warum jetzt:**
- Strukturell identisch mit Forst: Außendienst, Projektkoordination, Saisonarbeit
- `appfabrik-base` + `appfabrik-app-base` direkt übertragbar (60–70% Wiederverwertung)
- Forstmanager als Demo-Referenz direkt verwendbar
- Verbände gut zugänglich (BGL Bundesverband GaLaBau)

**Go-to-Market:**
1. Koch Aufforstung Case Study → GaLaBau-Verbände pitchen
2. Branchenspezifische Landing Page auf AppFabrik-Website
3. Direktakquise über LinkedIn bei Betrieben mit 10–30 MA

**Feature-Delta zu Forst (Neuentwicklung nötig):**
- Pflanzenkatalog (statt Baumarten)
- Regenrinnen-/Bewässerungsplanung
- Jahrespflege-Verträge (statt Einzel-Projekte)

### Rang 2: Handwerk & Bau — Score 3.65 🔴

**Empfehlung: Parallel zur GaLaBau-Akquise — mittelfristig**

| Kennzahl | Wert |
|---------|------|
| KMU-Betriebe DACH | ~650.000 |
| TAM | 1,2–2 Mrd. € |
| Ø Betriebsgröße | 5–50 Mitarbeiter |
| Digitalisierungsgrad | 🟡 Mittel (~35%) |
| Top Gewerke | Elektro, SHK, Maler, Dachdecker, Zimmerer |
| Wettbewerbsdruck | Mittel (viele Tools aber wenig White-Label) |

**Warum attraktiv:**
- Größter TAM aller 5 Märkte
- Viele Nischen mit spezifischen Anforderungen (jede Nische = eigenes Produkt)
- Zahlungsbereitschaft deutlich höher als Forst/Agrar

**Herausforderung:**
- Mehr Wettbewerb (Handwerk-Software ist bekannter)
- Komplexere Differenzierung nötig
- Gewerke-spezifische Features (SHK ≠ Elektro ≠ Maler)

**Empfohlene Einstiegsnische:** Zimmerer / Holzbau (Nähe zu Forst, White-Label noch nicht vorhanden)

### Rang 4: Agrar / Landwirtschaft — Score 3.65 🟡

**Empfehlung: Mittelfristig — nach GaLaBau-Erfolg**

| Kennzahl | Wert |
|---------|------|
| KMU-Betriebe DACH | ~220.000 |
| TAM | 300–500 Mio. € |
| Ø Betriebsgröße | 3–15 Mitarbeiter |
| Digitalisierungsgrad | 🔴 Sehr niedrig (<15%) |
| Besonderheit | Stark saisonabhängig, FKZ-Förderungen |

**Herausforderung:**
- Geringere SaaS-Zahlungsbereitschaft (Kostenbewusstsein)
- Förderung als Kaufanreiz nötig (Digitalisierungs-Fördergelder)
- GPS/GIS-Anforderungen komplexer (Feldgrenzen, Pflanzenschutz)

**Einstiegspunkt:** Lohnunternehmen (größer, zahlungskräftiger, mehr Außendienst)
